# Onenightwolf

一夜終極狼人的即時線上版本，與阿瓦隆共用同一個 Render service。

- 公開路徑：`/Onenightwolf/`
- WebSocket：`/ws/onenightwolf`
- 支援 3～10 人
- 牌庫固定為玩家人數加三張角色牌，支援推薦牌庫與自訂牌庫
- 角色包含化身幽靈、狼人、爪牙、守夜人、預言家、強盜、搗蛋鬼、酒鬼、失眠者、村民、皮匠與獵人
- 每局先擲 d100 決定順時鐘座位與玩家列表順序；夜間仍依固定角色順序進行
- 投票結束時以最終角色判定獵人；最終為獵人的玩家死亡後，其投票指向的玩家也會自動死亡，並支援獵人連鎖
- 夜晚持續顯示初始角色供玩家記憶，天亮進入討論後隱藏角色卡
- 夜間依初始角色決定叫醒順序與能力發動者；換牌會影響最終角色，但不會補發或取消夜間能力
- 夜間依角色階段進行；角色只在中央牌時會隨機等待 5～9 秒，牌庫未選該角色則直接跳過
- 重新整理後回到加入／重新連線大廳，可用保存的玩家身分重新連線
- 同一瀏覽器可保存多個玩家身分；玩家名稱輸入欄會即時比對同房間保存過的身分，輸入 A/B 這類曾使用名稱時，重新連線按鈕會切換成對應玩家
- WebSocket 斷線後，若本分頁曾進入房間，前端會自動用保存的玩家身分重新加入並同步狀態
