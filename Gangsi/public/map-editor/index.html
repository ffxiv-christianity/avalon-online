    Gangsi Map Format v1

    地圖編輯器說明

    基本操作

    • 使用工具列設定牆壁、封閉格、入口、地牢與寶藏。
    • 獵殺模式地圖另需放置機關 A 與機關 B。
    • 編輯內容會自動保存在目前使用的瀏覽器中。
    • 「復原」與「重做」可還原最近的編輯操作。

    牆壁

    • 單擊格線可放置或移除一段牆壁。
    • 從通道開始拖曳，可沿同一直線連續放置牆壁。
    • 從既有牆壁開始拖曳，可沿同一直線連續移除牆壁。
    • 拖曳只會影響起始牆段所在的同一條格線，不會修改交叉或相鄰格線。
    • 一次拖曳視為一次操作，可使用「復原」還原。

    地圖區域

    封閉格
    不屬於可通行道路，任何棋子都不能進入。
    入口
    所有冒險者開始遊戲時共同等待的位置。冒險者可從任一未被牆壁阻擋的相鄰出口進入地圖，離開後不能主動返回入口。
    地牢
    提燈怪開始遊戲的位置。被抓且仍有生命的冒險者也會回到地牢，並在下次行動時從任一合法出口重新進入地圖。
    機關／出口 A、B
    獵殺模式的人類從相鄰道路操作機關；完成前該格不可站立，完成後同一格轉為逃生出口，只有人類能移入並逃脫。
    • 每張地圖只能設定一個入口與一個地牢，兩者不能位於同一格。
    • 入口與地牢本身不是一般道路格，不能放置寶藏。
    • 相鄰且未被牆壁阻擋的道路格會自動成為出口；入口與地牢都至少需要一個合法出口。
    • 兩座機關不能與入口、地牢、封閉格、寶藏或彼此重疊。

    寶藏與驗證

    • 將寶藏標記拖曳到棋格上即可放置或移動。
    • 地圖需放置 A1–A5、B1–B4、C1–C4、D1–D5、E1–E5，共 23 個寶藏。
    • 同一格只能放置一個寶藏，且不能放在入口、地牢或封閉格。
    • 「地圖驗證」會提示缺少的設定、無法抵達的道路與其他錯誤。

    儲存與使用

    • 「下載地圖」會下載目前的地圖資料（.JSON）。
    • 下載完成後，請將 .JSON 檔案傳給網站作者，由作者加入可選地圖。
    • 載入內建地圖後仍可繼續編輯；完成修改後，建議先更改地圖名稱，再按「下載地圖」匯出自己的版本。
